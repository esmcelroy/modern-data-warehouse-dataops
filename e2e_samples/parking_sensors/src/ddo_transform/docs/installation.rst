============
Installation
============


Stable release
--------------

To install ddo_transform, run this command in your terminal:

.. code-block:: console

    $ pip install ddo_transform

This is the preferred method to install ddo_transform, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for ddo_transform can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/devlace/ddo_transform

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/devlace/ddo_transform/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/devlace/ddo_transform
.. _tarball: https://github.com/devlace/ddo_transform/tarball/master
